# Docker 
* Verbinden mit Container in bash: <br><pre>docker exec -it container_name bash</pre>

# Docker Compose
* Container starten:<br><pre>docker compose up -d</pre>
* Container stoppen:<br><pre>docker dcompose down</pre>
